Three Checks, One Report

Existence is table stakes. Claim-vs-source is the point.

A fabricated bibliography is rare. A real bibliography attached to invented specifics — a quote the source never contains, a figure it never reports, a conclusion it explicitly disclaims — is what actually shows up on a teacher's desk.

Check 1 · Deterministic

🔗 Cross-match

In-text citations matched against the works-cited list, both directions. Sources cited but never referenced. References that never appear in the text. Format sanity, stated as observations.

"Three works-cited entries are never cited in the text."

Check 2 · The paid value

⚖️ Claim vs. source

Does the source exist — and does it say what the essay claims? Quoted passages are searched in the reachable source text; figures are matched exactly; paraphrases are compared against the source's own words, both sides quoted.

"This quoted passage was not found in the source text."

Check 3 · Questions only

🔢 Data plausibility

Numbers and claims attributed to sources, sanity-checked for accuracy and plausibility. The most judgment-laden check, so it is phrased only as questions — never as findings.

"This 87% figure is attributed to a single-participant case study — worth asking about."

The Citations Report

Both sides quoted.
No verdict.

Every citation gets two factual reads — was the source located, and does it say what the essay claims — with the evidence basis stated on each. Paywalled and print-only sources fail lookup routinely, so the language is built to stay honest about what a search can and can't see.

✓ Located · Consistent

The source was found and the reachable text supports what the essay attributes to it. The evidence basis is stated — full text, abstract, snippet, or publisher description.

◐ Notable difference

The reachable source text differs from the essay's characterization — both sides quoted so you can judge. A quote not found in fully-reachable source text is the report's strongest finding.

◌ Access limited · Couldn't check

The source sits behind a paywall or in print, so the claim couldn't be compared. Stated as a fact about access — never as suspicion.

Honest Limitations

What Verify Citations can't tell you

Like every PaperTrail tool, this is a structured, repeatable data point — not a verdict. Three things to keep in mind before you weigh one.

🔒 The web has walls

Paywalled journals, print monographs, and regional databases are routinely unreachable. A legitimate source a student found in the school library can come back "Not located" or "Access limited."

→That's why those statuses are worded as facts about access — and why the report never says "fabricated." Ask the student to show you the source; a real one is easy to produce.

📄 Evidence has degrees

"Consistent with the abstract" is weaker evidence than "quote found on page 43." A paraphrase judged against a publisher's description is weaker still.

→Every claim read states its evidence basis — full text, abstract, snippet, or description — so you always know how much weight it can bear.

⚖️ Difference is not deception

A "Notable difference" can be sloppy paraphrase, an honest misreading, a secondary source quoted secondhand — or something worth a longer conversation. The report can't tell you which.

→Each finding comes with a suggested question. A student who genuinely read the source answers it in seconds — that answer is the real data point.
