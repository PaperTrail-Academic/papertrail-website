3. Description of Service

PaperTrail Academic provides Chrome browser extension tools for writing analysis and authorship consistency review within Google Docs. The Service includes:

  • PaperTrail Inspect: Revision history analysis, writing session reconstruction, and behavioral pattern detection based on Google Docs revision data. Core Inspect features are available to all users without an account. Paste analysis and struggle moment detection are Inspect features that require an active StyleMatch subscription to access.
  • StyleMatch: Local, on-device stylometric comparison of two writing samples across eight metrics to assess authorship consistency. No text is transmitted when using StyleMatch. Users who create a free account receive three complimentary StyleMatch runs. Unlimited StyleMatch access requires an active subscription.
  • Verify: AI-assisted writing analysis that transmits submitted text to a third-party AI provider (Anthropic, Inc.) for processing. Available as a Consistency Report (single sample) or a Baseline Deviation Report (comparative, requires a StyleMatch report as input). See Section 8 for data handling details.

8. Data, Privacy, and Student Text

StyleMatch (on-device): All stylometric computations are performed locally in your browser. No text you enter into the StyleMatch panel is transmitted to PaperTrail Academic or any third party. This feature is privacy-safe by design.

Verify (AI-assisted analysis): When you run a Verify report, the text you submit is:

  1. Transmitted to PaperTrail Academic's secure backend (Supabase, hosted on AWS infrastructure)
  2. Forwarded to Anthropic, Inc.'s API for AI-assisted analysis
  3. Returned as a structured report and saved to your account's report history

Text submitted through Verify is not stored by PaperTrail Academic beyond the metadata required to save your report (report ID, scores, timestamps, and any student name or assignment label you enter). The full essay text is not retained by PaperTrail after the report is generated.

Anthropic processes submitted text under its commercial API terms, which prohibit using API inputs for model training. As of September 2025, Anthropic retains API inputs for 7 days before automatic deletion. See anthropic.com/privacy for their full policy.

Your responsibility. You are responsible for ensuring that your use of Verify complies with:

  • Your institution's data governance and acceptable use policies
  • Applicable education privacy law in your jurisdiction (including but not limited to FERPA in the United States and GDPR in the European Union)
  • Any consent or notice requirements your institution has established for processing student work

Where your institution's policy requires it, you should anonymize student-identifying information from text samples before submitting them through Verify.
